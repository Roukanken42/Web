Bloodlust

Bloodlust effect (or Execution Damage), is a effect that will increase the damage if your skill based in enemy HP

So far the only know skills that have this effect are:

  • Archer: Radiant Arrow, Penetrating Arrow
  • Berserker: Thunderstrike
  • Sorcerer: Meteor, Nova, Void Pulse, Arcane Pulse
  • Reaper: Sundering Strike
  • Brawler: Haymaker
